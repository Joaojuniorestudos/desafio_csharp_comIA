# Exercício 04 — Controle de Estoque com Dictionary

## Cenário real

Você precisa controlar o estoque básico de uma loja, atualizando quantidades de produtos de forma rápida.

## Objetivo

Criar um programa em C# que:

1. Use um `Dictionary<string, int>` para guardar **Produto -> Quantidade em estoque**.
2. Permita cadastrar produto com quantidade inicial.
3. Permita dar entrada e saída de unidades de um produto.
4. Impedir que a saída deixe o estoque negativo.
5. Exiba um relatório final com produto e quantidade atual.

## Como executar

```bash
dotnet run
```
