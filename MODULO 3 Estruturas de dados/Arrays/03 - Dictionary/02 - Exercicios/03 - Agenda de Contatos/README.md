# Exercício 03 — Agenda de Contatos com Dictionary

## Cenário real

Você está criando uma agenda simples para cadastrar e consultar contatos rapidamente pelo nome.

## Objetivo

Criar um programa em C# que:

1. Use um `Dictionary<string, string>` para guardar **Nome -> Telefone**.
2. Permita cadastrar novos contatos.
3. Permita buscar um contato pelo nome e mostrar o telefone.
4. Liste todos os contatos cadastrados.
5. Mostre mensagem quando o contato não existir.

## Como executar

```bash
dotnet run
```
