# Exercício 02 — Maior e Menor Valor em Array

## Cenário real

Você está desenvolvendo um relatório simples que precisa encontrar rapidamente o maior e o menor valor de uma lista de dados.

## Objetivo

Criar um programa em C# que:

1. Leia uma sequência de números inteiros do usuário.
2. Armazene esses valores em um `int[]`.
3. Encontre o **maior** e o **menor** valor do array.
4. Conte quantas vezes o maior e o menor aparecem.
5. Mostre essas informações no console (maior, menor e quantidades).

## Como executar

```bash
dotnet run
```
