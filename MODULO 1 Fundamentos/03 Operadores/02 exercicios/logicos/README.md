## Exercício 04 – Acesso a promoção do clube

Crie um programa para um clube que queira verificar se uma pessoa **tem direito a uma promoção**.

O programa deve solicitar:

- Nome da pessoa.
- Idade (inteiro).
- Se é sócio do clube (`true` ou `false`).
- Se tem convite especial (`true` ou `false`).

Use **operadores lógicos e relacionais** para:

- Verificar se a pessoa **pode participar da promoção** quando:
  - for **maior ou igual a 18 anos** **e** for **sócio**, **ou**
  - possuir um **convite especial**, mesmo não sendo sócio.

No final, exiba:

- O nome da pessoa.
- A idade.
- Se é sócio.
- Se tem convite especial.
- Uma mensagem dizendo se **tem direito à promoção** ou **não tem direito à promoção**.

Tente usar os operadores:

- `>=`
- `&&`, `||`
- `!`
